# precise-warmth

A design sensibility skill for AI coding agents — ZUENS2020's aesthetic profile.

**Core idea:** A solitary individual moving with precise, restrained warmth through a vast, highly structured, technological system.

## What This Skill Does

When loaded, this skill guides AI agents to make creative and design decisions aligned with an industrial-minimalist, geometrically-ordered, precision-warmth aesthetic.

**Domains covered:** visual design, UI, architecture, film, sound, hardware, narrative tone.

## Install

```bash
npx skills add ZUENS2020/precise-warmth
```

Or install manually by placing `SKILL.md` in your agent's skills directory.

## When It Activates

- Automatically on creative/design projects
- When user mentions: aesthetics, design style, visual taste, art direction, creative direction, look and feel
